Gallery of Hybrid Models
========================

This page consists of the hybrid AI models generated from the ACTM performers. The page consists of training code, prediction approaches and model weights. These models are tailored for a specific applications within the program (e.g. a specific tipping point like AMOC, a model for a climate intervention scenario like marine cloud brightening, etc.). Please reach out to the respective authors if there is an interest for collaboration or expansion beyond the current scope of application.
